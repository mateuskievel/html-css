Formatações de Texto

Negrito

Para dar destaque em um termo, deixando-o em negrito, utiliza-se a tag <strong>.

Itálico

Para deixar um termo em itálico, utiliza-se a tag <em>.

Marcação

Para marcar uma frase, utiliza-se a tag <mark>.

E para trocar a cor da marcação, devemos utilizar um parâmetro de CSS - o style -, dentro da tag mark.

Exemplo: <mark style="background-color: rgb(77, 209, 209);">.

Atenção!

Utilizar o parâmetro style dentro da tag é uma forma ineficaz quando temos diversas marcações. Um método para definir a cor da marcação dentro desta página é criando uma tag style no cabeçalho (head).

Exemplo:

  <head>
      <style>
          mark {
              background-color: rgb(77, 209, 209);
          }
      </style>
  </head>

Letras Grandes e Letras Pequenas

Letras Grandes

A tag para deixar uma frase em uma fonte maior caiu em desuso, mas era representada pela tag <big>.

Letras Pequenas

Já para deixar em uma fonte menor é utilizada a tag <small>.

O porquê a tag <big> caiu em desuso mas a <small> continua ativa não têm uma resposta exata, mas eu concordo com o que o Professor Guanabara teorizou. A tag <small> continua em uso pois ela tem sentido, pois ela remete às letras pequenas vistas em contratos, não é apenas um estilo.

Texto Excluído ou Texto Riscado

Para indicar um texto que fazia parte do documento mas que agora deve ser desconsiderado, ou seja, não está mais em vigor, devemos utilizar a tag <del>.

Texto Inserido ou Texto Sublinhado

Para dar ênfase em um texto que foi incluido posteriormente, que passa a ser considerado agora, utiliza-se a tag <ins>.

Texto Sobrescrito

Para fazer com que um texto fique sobrescrito, ou seja, em uma letra pequena alinhada na parte superior da linha, para formar, por exemplo, uma fórmula como a x20+3, utiliza-se a tag <sup>.

Texto Subescrito

Já para formar um texto em que a letra fica pequena e alinhada na parte inferior, para formar, por exemplo, a fórmula H20, utiliza-se a tag <sub>.
