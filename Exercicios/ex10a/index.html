Usando Links

Tag

A tag utilizada para indicar onde será a "zona" clicável para levar ao novo site é a <a>.
Dentro da tag <a> temos o parâmetro href que é onde colocamos a URL da página para qual o usuário será redicionado.

Links Externos

Um link externo é usado para redicionar o usuário para outro link que não faz parte do seu site.
Por exemplo:

Ao clicar aqui você irá acessar meu perfil no AniList!

No caso de uma URL externa, após o parâmetro href também utilizamos o parâmetro target junto do atributo _blank, para assim dizermos que ao clicar no link, uma nova página será aberta em uma nova aba.

Além disso, também usamos o parâmetro rel com o atributo external. Esse conjunto não tem uma função visível ao usuário, mas ela serve para dizer aos mecanismos de busca que o link colocado é de uma fonte externa, ou seja, de outro site.

nofollow

O atributo _nofollow é utilizado com o parâmetro rel quando não queremos que o mecanismo de busca relacione o nosso site diretamente com aquele site externo.

Isso é feito quando não temos total confiança ou certeza do que será aquele link externo, então usamos este atributo geralmente para redicionamento de links de anunciantes ou de uma fonte não confíavel.

Porém não confunda! Não confíavel não se refere a colocar links maliciosos, e sim sobre sua credibilidade, por exemplo, para uma página de uma notícia a qual o conteúdo pode acabar sendo falso ou tendecioso.

Links Internos

Links internos são usados para redicionar o usuário para outra página do seu site.

Assim como nos links externos, utilizamos o parâmetro target, porém com o atributo _self, fazendo com que a página seja aberta na aba atual, sobrepondo a página anterior.

E para o parâmetro rel utilizamos ou o atributo next ou prev, que irá indicar que a página que será acessada deve ser considerada uma página seguinte a atual (next), ou uma página anterior a atual (previous).
